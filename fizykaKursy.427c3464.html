An online course store. First big, fullstack project I implemented mostly on my own, asking my co-worker / mentor for advices and verifying ideas during development.

Project built with PHP (CakePHP), MySQL, Apache, jQuery, CSS. It included a bunch of features like:

  • CMS / Admin panel
  • online payments integration (tpay)
  • protecting videos from being downloaded via browser extensions
  • 3rd party authorization via twitter and facebook
  • front-end design
  • ...many others
Huge experience earned by this project. Especially happy to see it up and running for so many years.
