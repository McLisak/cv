A suite of sport specific data presentation embeddable apps.

A monorepo project containing node backend and several sport specific react clients.

Each client is meant to be embedded on particular operator website, which is achieved by a library the operators use for easily configurable embed implementation and two way communication.

Each operator has possibility to have their own custom theme applied to embedded app and they can place the embed in desired place on their site thanks to each app auto adjusting its layout to given available space.
