    Where I've been

    01/2016 - 02/2017

    180Creative - Full Stack

    Full Stack

    Tech stack: PHP, Apache, MySQL, JS, jQuery, angular.js (yes the first one), CSS, SCSS, git

    First job. Creating web apps and websites for client. Each dev in the team was working mostly alone on his own projects. The process was far from perfect and the tempo was very fast. As a newbie this was a stressful, but an amazing experience. High tempo and lots of challenges helped me learn web development rapidly fast. I've earned enough knowledge to polish afterwards.

    09/2017 - 11/2018

    Perform - Spieler

    Front End

    Tech stack: JS (vanilla), Mocha, CSS, HTML Video, Streaming libraries (hls, dash), Video Ads (Google IMA), git

    Joined Perform - Spieler project. We were building a core video players that other teams could further customize and use in their projects. The "spieler" library was meant to solve common cross-browser problems that different teams were facing during their development. It had to expose unified simple API, handle different source formats, ads, provide customizable features and "work everywhere".

    11/2018 - 05/2020

    DAZN Media (ex Perform) - Commercial

    Front End

    Tech stack: TS (vanilla), Vue, Vuex, CSS, Jest, Drone, Video Ads (Google IMA), git, wdio, TODO

    Migrated to newly created Commercial team. The idea for the team was to be an "ads experts team" handling technical aspect of the ads across Perform/DAZN Media portals. Before I left we created 3 useful products for internal usage that helped other teams a lot:

    DUX - a modular library (read more TODO), CTE - a web app that improved ad template management, VAF (included later in DUX) - a VideoAdFramework - like Spieler but for ads-only :)

    Building those entirely from scratch was a challenging but very pleasing job.

    05/2020

    DAZN - Living Room

    Software Engineer / Full Stack

    Tech stack: TS (vanilla), React, Jest, CSS, LivingRoom Platforms, Drone -> GithubActions, AWS, git, wdio, gherkin
